# Statistical and Machine Learning Analysis of DC Housing Prices

## Overview
This project evaluates a wide range of **statistical and machine learning models** to predict residential property prices in Washington, DC. The analysis compares classical regression, classification models, and modern machine learning techniques.

## Models Used
- Linear Regression
- Polynomial Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Linear & Quadratic Discriminant Analysis (LDA/QDA)
- Ridge & LASSO Regression
- PCR & PLS
- Regression Trees
- Random Forests

## Data
- Source: Kaggle DC Residential Property dataset
- Extensive data cleaning and filtering
- Training/testing split for predictive evaluation

## Key Insights
- Polynomial regression improved over basic linear models
- Tree-based models captured non-linear interactions
- Shrinkage methods highlighted variable importance
- Classification accuracy exceeded 80% in several models
